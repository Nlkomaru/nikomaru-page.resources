# コンテナとは

コンテナはアプリケーションとランタイム環境をパッケージ化する技術です。

これにより、コンテナ内のアプリケーションを複数の環境間で容易に移行することができます。

また、osの機能の一部を共有することで、コンテナは軽量で高速に起動することができます。
